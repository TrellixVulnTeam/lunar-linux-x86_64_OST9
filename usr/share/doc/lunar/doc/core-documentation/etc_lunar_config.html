File Name: /etc/lunar/config


Contains the initial defines required for the entirety of the Lunar core toolset's operation.

The following tables contain a list of global variables defined for use throughout the Lunar tool set. These are used by /sbin/lget,/sbin/lin,/sbin/lrm,/sbin/lunar, and /bin/lvu and many of the subroutines and functions which make up the toolset.

Environmental Defines


Define

Explanation
DIALOGRC=/etc/lunar/dialogrc Configuration file for the dialog binary used by the lunar core tools.
